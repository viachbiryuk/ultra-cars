# Ultra cars

## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker-compose up
```

## Seed

```bash
# npm run seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
(make sure DB postgres on localhost:9002 before run)
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
